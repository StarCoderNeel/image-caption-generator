# Image Caption Generator

An AI-powered image captioning tool that generates descriptive text for images, useful for accessibility and content creation. The project demonstrates AI model integration and natural language processing.

## Features

- Image upload and processing
- Caption generation with NLP
- Result visualization and download

## Installation

```bash
# Clone the repository
git clone https://github.com/{username}/image-caption-generator.git
cd image-caption-generator

# Install dependencies
pip install -r requirements.txt
```
